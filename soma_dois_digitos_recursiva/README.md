### Soma dos Dígitos Recursiva

Descrição:
Dado um número inteiro positivo, implemente uma função que calcula a soma recursiva dos dígitos até que o resultado tenha apenas um dígito. Este problema é conhecido como "Soma dos Dígitos Recursiva" ou "Número Digital".

Exemplo:
Para o número `9875`, a função deve retornar `2`, pois:

`9 + 8 + 7 + 5 = 29`

`2 + 9 = 11`

`1 + 1 = 2`


Entrada de Exemplo:
`9875`

Saída de Exemplo:
`2`
